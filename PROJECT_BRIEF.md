## 12. Accessibility & Compliance

- **Standard:** WCAG 2.2 AA
- **Languages:** Arabic (RTL) primary + English (LTR) secondary — native experience in both
- **Keyboard:** Full keyboard navigation — animations play correctly for keyboard users
- **Screen Readers:** Semantic HTML, ARIA labels, proper heading hierarchy; tested in Arabic on iOS VoiceOver and NVDA
- **Motion:** Respect `prefers-reduced-motion`; LCP elements (hero H1) always visible on first paint
- **Contrast:** WCAG AA maintained — brand red buttons use `#B03020` (5.1:1 with white text)
- **Forms:** Clear labels above inputs, accessible multi-step form with fieldset/legend and aria-live announcements

### Cookie Consent & Data Privacy

- **Cookie consent:** CookieYes CMP installed before any analytics scripts load
- **Compliance:** GDPR (EU visitors) + Egypt Data Protection Law No. 151/2020
- **Scope:** All tracking cookies (GA4, Meta Pixel, Microsoft Clarity) blocked until user grants consent
- **Google Consent Mode v2:** GA4 fires in limited consent mode before explicit consent
- **Cookie Policy:** Linked from CookieYes banner and footer navigation
